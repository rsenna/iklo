# `refs/` — reference material & historical snapshots

Not Iklo code. Not tests. Not active documentation of Iklo. Three kinds of
content live here:

```
refs/
  ucblogo/                     → UCBLogo (Berkeley Logo) — historical influence
  netlogo/                     → NetLogo user manual — reference material
  clock-calculus-summary.md    → clock calculus (Pouzet) distilled, with Iklo-specific
                                  relevance notes; source PDF kept local, not committed
  AGENTS.old.md                → historical snapshot of the pre-split AGENTS.md
  README.old.md                → historical snapshot of a previous README.md
  tour.old.iklo                → historical language tour (source of the sigil taxonomy)
```

The `*.old.*` files are frozen — do not edit. They exist so we can trace the
provenance of design decisions. Single-file summaries like
`clock-calculus-summary.md` distill an external, non-Iklo source and add
Iklo-specific analysis — they are original writing, not reproductions, and
may be extended. See [`../LANGUAGE.md`](../LANGUAGE.md) for what Iklo
actually is today. Iklo is *not* a Logo — the Logo influence is stylistic
and historical.
