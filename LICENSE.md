## Scope in plain language

- Copyleft applies to code in this repository that implements Iklo.
- If you write code *in Iklo* (applications, scripts, or Iklo libraries), that
  code is not forced under GPL solely because you used Iklo.
- Files/output generated by Iklo are not forced under GPL solely because Iklo
  generated them.
